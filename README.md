# First Builder

Exercice de formation

## Prérequis

Linux ou mac.

Installer Git, Node.js et NPM.

Ajouter `./node_modules/.bin` à son path.

## Etapes

1. Faire un Fork ce ce projet
2. Cloner son fork
3. faire un `npm init` 
4. Ecrire la commande pour compiler le scss vers le donnier `./build/css/`
